Transparent by design

How this experiment works

  1. Local scanReads signals already available to the page. It does not call an outside lookup service.
  2. Network lookupOnly after approval, contacts ipapi.co, echo.websocket.events, and Google’s public STUN service. Each service can observe the request’s public IP.
  3. Permission requestsLocation, camera, microphone, and notifications are requested individually through your browser.

The web server still receives the normal request needed to load this page. The generated report itself is kept in memory in this tab and is not uploaded by this demo.
