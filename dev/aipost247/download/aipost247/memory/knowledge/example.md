# Example knowledge file

Drop facts, product details, FAQs, or reference material into this folder as
`.md` or `.txt` files. AIPost247 reads them and uses them as context so posts
are accurate and on-topic.

You can add as many files as you like, e.g.:

    memory/knowledge/products.md
    memory/knowledge/faq.txt
    memory/knowledge/upcoming-events.md

Replace this example with your own content (or delete it).
